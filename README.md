# Site Web d'AtWork

[Clickez ici pour accéder au site Web]()

## Installation

### Prérequis
* [Flutter](https://flutter.dev/)
* [Python 3](https://www.python.org/downloads/)

### Commandes en local
```
flutter build web --web-renderer html
cd build/web
python3 -m http.server 8000
```
